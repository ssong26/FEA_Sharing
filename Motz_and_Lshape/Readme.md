# Motz Problem and L-Shaped Singularity FEM Examples

Finite element examples for classical singularity problems in computational mechanics.

This folder contains MATLAB implementations for two benchmark problems widely used in finite element analysis (FEA) and numerical PDE research:

- Motz problem
- L-shaped domain singularity problem

These problems are classical test cases for studying:

- stress singularities
- corner singularities
- convergence behavior
- adaptive meshing
- numerical accuracy near singular points

---

# Background

Singular problems are important benchmarks in computational mechanics because the solution gradients become unbounded near corners or mixed boundary-condition junctions.

Standard FEM convergence may deteriorate significantly near these singular regions.

These examples were developed to explore:

- mesh refinement strategies
- singularity behavior
- finite element convergence
- numerical treatment of sharp-gradient fields

---

# 1. Motz Problem

The Motz problem is a classical Laplace-equation benchmark with mixed boundary conditions.

It is frequently used to evaluate FEM performance near singular points generated by boundary-condition transitions.

## Governing Equation

The problem solves:

$$
\nabla^2 u = 0
$$

with mixed Dirichlet and Neumann boundary conditions.

The singularity appears at the junction where the boundary conditions change type.

## Features

- Laplace equation FEM solver
- Mixed boundary conditions
- Corner singularity analysis
- Triangular finite elements
- Sparse matrix assembly

## Numerical Challenges

Near the singular point:

$$
u \sim r^{\lambda}
$$

where:

- $r$ is the distance to the singularity
- $\lambda$ is a non-integer singular exponent

The solution gradient becomes highly localized, making the problem a useful FEM benchmark.

# 2. L-Shaped Domain Problem

The L-shaped domain problem is another classical benchmark for singularity analysis.

The re-entrant corner produces stress and displacement singularities that strongly affect FEM convergence.

## Geometry

The domain contains a re-entrant corner with interior angle:

$$
\theta = \frac{3\pi}{2}
$$

which generates singular behavior near the origin.

## Governing Equation

The governing equation is:

$$
\nabla^2 u = 0
$$

with prescribed boundary conditions on the outer boundaries.

## Features

- Unstructured triangular mesh
- Re-entrant corner singularity
- FEM convergence study
- Visualization of localized fields

---

# Numerical Method

The implementation uses:

- Linear triangular finite elements
- Gaussian integration
- Direct stiffness assembly
- Sparse matrix solution

The elemental formulation is based on:

$$
K_e =
\int_{\Omega}
(\nabla N \cdot \nabla N)\, d\Omega
$$

where:

- $N$ denotes the shape functions
